# Kisan Alert — Landing Page

Modern glassmorphism landing page for Kisan Alert, an AI-powered agriculture assistant.

## Stack
- React 18 + Vite
- Tailwind CSS
- lucide-react icons

## Run
```bash
npm install
npm run dev
```
Open the printed local URL (usually http://localhost:5173).

## Build
```bash
npm run build
npm run preview
```

## Structure
```
src/
  App.jsx
  main.jsx
  index.css
  hooks/useReveal.js
  components/
    Navbar.jsx
    Hero.jsx

    HowItWorks.jsx

```
